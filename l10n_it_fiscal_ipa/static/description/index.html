IPA Code and Destination Code in Partner Record / Codice IPA (IndicePA)

IPA Code

This module adds IPA (IndicePA) code and Recipient Code fields to partner, used by Italian Electronic Invoice.

http://www.indicepa.gov.it

Codice IPA (IndicePA)

Questo modulo permette l'inserimento del codice IPA (IndicePA) e del Codice Destinatario nell'anagrafica cliente.

Questi dati sono indispensabili per la gestione della Fattura Elettronica B2B e per la FatturaPA.

http://www.indicepa.gov.it
