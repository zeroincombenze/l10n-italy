Enhanced Odoo Features / Funzioni evolute Odoo

This module add various useful features to Odoo instance in order to make installation interface indipendent by version.

Questo modulo aggiunge varie caratteristiche utili che rendono l'interfaccia dell'installazione indipendente dalla versione di Odoo.
